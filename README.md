# Việt Hóa Where Winds Meet (aka Yến Vân Thập Lục Thanh)

Đây là bản việt hóa, lấy gốc từ bản nội địa TQ.

## Cài đặt

1. Tải file ~~`vi.zip`~~ `translate_words_map_en.zip` việt hóa [tại đây](https://github.com/DichGameChoi/wwm-vn/releases) hoặc vào thẳng Releases page
2. Giải nén file được nội dung như sau

<img width="262" height="153" alt="img-1" src="https://github.com/user-attachments/assets/004a408a-abfa-4b9d-b2f4-ed0765cac8d1" />

3. Đè 2 file này vào thư mục `locale` của game.
   
Lưu ý nên để game check resource xong thì đè file.

## Lưu ý

- Chờ game kiểm tra tài nguyên xong thì đè file.
- Để dùng được tiếng việt thì cài đặt sử dụng English.
- Tác giả của bản dịch này không chịu bất kỳ trách nhiệm nào liên quan đến tài khoản của bạn.

## Cách tìm thư mục locale

- Mục tiêu cần tìm thư mục tên `oversea` -> `locale` hoặc `locale` để đè file.

- Có thể mở thư mục cài Game tìm thư mục: `Package` -> `HD` hoặc `LocalData` -> `Patch` -> `HD`
